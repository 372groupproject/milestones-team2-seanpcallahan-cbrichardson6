opendoor.f95

for Unix:
  - nagigate to the directory with the file
  - check that you have the gfortran compiler program available by typing gfortran --version
  - on the command line call 'f95 -o opendoor opendoor.f95' to compile it
  - run the program with ./opendoor
 
 for Windows
  - nagigate to the directory with the file
  - check that you have the gfortran compiler program available by typing gfortran --version
  - compile the program with 'f95 opendoor.f90 -o opendoor.exe'
  - run it
